# Small-scale Technology Certificate (STC) Intake Platform

This repository tracks the design for a web platform that lets solar panel and battery installers assign their rights to create Small-scale Technology Certificates (STCs) to your company. The platform focuses on collecting compliant installation data, validating it against Clean Energy Regulator (CER) requirements, and capturing supporting evidence (photos and video) for streamlined STC creation.
